# Lyrics Video Maker

An automation tool built in Python that generates lyrics videos given the artist name and the song name.
Demonstration:

1. Maroon 5 & Cristina Aguilera - Moves Like Jagger

2. Lady Gaga & Bruno Mars - Die With A Smile
